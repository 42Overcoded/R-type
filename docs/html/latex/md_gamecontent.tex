if you want the enemy to shoot, go to the \textquotesingle{}config\+Files/bullet.\+json\textquotesingle{} and add a projectile with a tag, a hitbox, a sprite, a speed and more ...

to animate the enemy, go to the animate\+\_\+enemy function in the \textquotesingle{}src/game\+Engine/game/game\+Logic/animation\textquotesingle{} and add a case with the tag of your enemy and the animation you want to play. the code will handle the rest.\hypertarget{md_gamecontent_autotoc_md26}{}\doxysection{To add miscellaneous content\+:}\label{md_gamecontent_autotoc_md26}
